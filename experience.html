Swift Navigation - Software and Estimation Intern

San Francisco, CA, USA, June - August 2016

During my time at Swift Navigation I was working in the Estimation team. I was set to reasearch algorithms to detect up to multiple satellite signal errors and then implement it in their Python protoyping enviroment. I successfully implemented a method to detect when signals were statisically unlikely by calculating the chi squared statistic from the residual terms and the residual covariance matrix and then testing it against a threshold calculated from the given accepted probability of false alert.

I was also set to develop an algorithm to detect cyle slips in the carrier phase tracking. By using calculated doppler shift values from single differencing carrier phase measurements, I could use an iterative least squares to estimate the velocity and clock drift shift, and the residual. From that I could use some of the same outlier detection algorithms from before to decide which signals were likely to originate from cycle slips. The algorithm I developed was able to sucessfully correctly detect and exclude several cycle slips simultanously with a very low probability of false alerts and wrong exclusions.
